# Personal Portfolio Website

## Description
This is a Personal Portfolio Website developed using Flask, HTML, and CSS.

## Features
- Home Page
- About Me
- Skills Section
- Projects Section
- Education Section
- Contact Section

## Technologies Used
- Python
- Flask
- HTML
- CSS

## How to Run
1. Install Flask:
   pip install flask

2. Run the application:
   python app.py

3. Open your browser and visit:
   http://127.0.0.1:5000/
